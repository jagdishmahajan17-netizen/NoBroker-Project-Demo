# Risk Register — Property Listing Project

|---------|------------------------------------|--------|------------|-------------------------------------|
| R1      | Fake property listings             | High   | High       | Verification team + automated checks|
| R2      | Duplicate property entries         | Medium | High       | Unique property ID enforcement      |
| R3      | Image upload abuse (large/invalid) | Medium | Medium     | Validation rules + size limits      |
| R4      | Backend downtime                   | High   | Low        | Redundancy + monitoring alerts      |
